## Installation
Add this line to your application's Gemfile:

```ruby
gem 'universal_ar'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install universal_ar
```
```bash
$ rails g universal_ar:install
$ rails g devise User
$ rails generate devise:controllers users
$ rails generate devise:views users
$ rails db:migrate
```
